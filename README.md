## Introduction

Lors de mon passage à l'Institut Universitaire de technologie du Limousin au département Informatique, j'ai eu un module (m2201) qui avait pour objectif de nous donner les prémices de l'architectur systeme et les mécanismes de programmation d'un système informatique. Dans ce module nous avons abordé l'encodage dans différentes bases (binaire, octal, decimal, hexadecimal). Par la suite, le cours m'a permis d'apprendre le principe de la Stack (pile) et des registres avec un programme en Rust et assembleur. Cependant, le nombre de cours sur le module étant restrinct je n'ai pas pus apprendre beaucoup plus de choses sur les langages d'assembleur. Afin d'éclaircir le flou de lié à certaines notions et d'approfondir le cours, j'ai pris la décision d'apprendre un langage d'assembleur et de developper de petit programme, en ayant comme objectif principal de comprendre cet manière de programmer.
